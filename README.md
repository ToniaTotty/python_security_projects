# python_security_projects

## Overview
This repository contains a collection of Python-based cybersecurity projects focused on security automation, log analysis, and basic threat detection. The projects are designed to simulate real-world SOC (Security Operations Center) tasks.

## Purpose
To develop practical cybersecurity skills using Python for:
- Log analysis
- Threat detection
- Security automation
- Basic incident identification

## Projects Included
- Password Strength Checker
- Log Analyzer (Brute-force detection)
- Port Scanner
- URL Reputation Checker
- File Integrity Checker

## Skills Demonstrated
- Python scripting
- File handling
- String parsing
- Networking basics
- Security logic development

## Tools Used
- Python
- VS Code
- Linux (basic environment)
